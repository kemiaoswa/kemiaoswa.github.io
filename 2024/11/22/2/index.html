🆔脚本名称：CSDN Focus

⭐️ 脚本功能：CSDN 优化

📁 脚本简介：一个用于优化 CSDN 网站的脚本，支持黑暗模式，能够去除页面中的弹窗、广告和其他干扰元素，提供一个干净的阅读环境。

还可以自动展开文章和评论，方便用户查看内容，并允许自由复制文本。

🗂 NotallyX - 简约开源 Android 笔记应用

▎ 软件功能：笔记工具

▎ 软件平台：#Android

▎ 软件介绍：一个开源笔记应用，支持创建富文本笔记，具备加粗、斜体等文本格式化功能，还可以添加任务列表及子任务。

它支持附加图片、PDF 等文件类型，允许用户通过标题、修改时间或创建时间对笔记排序，使用颜色、标签和置顶功能便捷管理笔记。

还支持点击链接，提供撤销 / 重做和快速分享功能 ，支持笔记加密、配置自动备份，并提供导入 Google Keep 和 Evernote 数据的功能。

▎ 软件下载：点击下载

🤖 NGCBot - 基于 HOOK 机制的微信机器人

▎ 项目功能：微信机器人

▎ 项目介绍：一个开源功能强大的微信机器人，支持安全新闻定时推送、天气查询、手机号归属地查询、星座查询等多种服务，自定义程度丰富，小白也可轻松上手。

此外还包括积分系统、自动拉人进群、广告检测和 AI 回复等功能，还支持多种 AI 接口，如 Gpt 和星火，用户可以根据需要配置相应的 API 密钥以启用特定功能。

▎ 项目地址：点击下载

⬇️ so-novel - 开源交互式小说下载器

▎ 软件功能：小说下载

▎ 软件平台：#Windows #macOS #Linux

▎ 软件介绍：一个开源交互式小说下载工具，无需安装，只需解压即可使用。

功能包括通过书名或作者搜索小说，并将小说以 epub、txt 或 html 格式导出，方便用户导入喜欢的阅读器进行阅读 。

⭐️ 软件功能：音乐播放器

➡️ 支持平台：#Android #iOS

📁 软件简介：一款功能强大的音乐播放器，不仅支持多种音频格式，如 MP3、WAV 和 AAC，还可以根据自己的喜好定制播放列表和界面。

创作者可以通过 Winamp 播放自己的作品，并借助其分享功能，将音乐分享给更广泛的听众，增加曝光率。

⭐️ 项目功能：Markdown 编辑器

🆔 软件名称：VOX

⭐️ 软件功能：音乐播放器

➡️ 支持平台：#Windows #macOS

📁 软件简介：一款高保真音乐播放器，支持多种音频格式，包括无损格式，能够满足音乐爱好者对音质的高要求。

可以通过 VOX 轻松管理音乐库，进行在线音乐流媒体播放，并享受高分辨率音乐的无缝体验。

还提供云存储服务，可以将音乐上传到云端，随时随地访问自己的音乐收藏。

🆔 软件名称：Kalmia

⭐️ 软件功能：文档内容管理系统

➡️ 支持平台：#Windows #macOS #Linux

📁 软件简介：一个用 Go 语言编写的文档内容管理系统（CMS），具有多用户支持和版本管理功能，允许用户轻松编辑和维护文档。

集成了 Markdown 编辑器，使得文档的创建和修改变得简单快捷。支持与 RsPress 文档的管理，适合需要处理大量文档的团队和个人使用。

🆔 网站名称：zgqinc

⭐️ 网站功能：书源、图源、订阅源

📁 网站简介：一个大型资源整合平台，主要提供书源、图源、订阅源、规则、直播源等多种资源的下载和导入功能。

可以通过一键导入或本地导入的方式，轻松获取和管理各种资源。网站上提供了 24767 个书源、2695 个 RSS 订阅源、335 个 TTS 朗读引擎、189 个净化规则等。
